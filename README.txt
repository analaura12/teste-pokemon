Olá! 

O projeto foi feito em React.js.

Utilizei o método fetch para retornar as informações da API.

Para acessar o projeto, é necessário abrir a pasta raiz com o terminal e executar o seguinte comando:

- npm install (requer node.js);

Isso instalará todas as depências que foram usadas no projeto.

Para executar o projeto, digite no terminal: 

- npm start
